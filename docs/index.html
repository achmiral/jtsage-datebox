DateBox is a jQueryMobile / Bootstrap / jQueryUI plugin that aims to make user interaction with dates and times simple and intuitive. It is a collaborative work, with a full range of features allowing easy implementation, and painless extensibility. New to version 4 is the inclusion of Bootstrap and jQueryUI operation, along with a new build system that makes adding the framework of your choice as easy as possible.
Upgrade Note: DateBox4 is backwards compatible with jQM-DateBox, however the namespace for the widget has changed. Any references in your own code to mobile.datebox or mobile-datebox must be updated to jtsage.datebox or jtsage-datebox.
Regressions / Maintainer Note: So, the reality is, it looks like development on jQM has pretty much ended. The news for jQueryUI doesn't look much better these days. The bootstrap versions *require* the jQueryUI widget code (see the install docs). This likely won't change. But, here is what this all means: The maintainer does not actively use the jQM or jQueryUI versions - expect regressions and visual bugs. When they are reported, he will fix them. Same probably with Bootstrap 3, as all of his projects (mostly) have moved to BS4.
